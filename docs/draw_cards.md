# 🔮 插件指令

> [!WARNING]
> 使用插件指令时，需要加上 NoneBot 配置的指令前缀（默认为 `/`）

## 基本指令

### 抽卡

- 指令：`抽卡`
- 功能: 进行10连抽卡

### 抽卡帮助

- 指令：`抽卡帮助`
- 功能: 获取插件基本帮助

### 抽卡排行

- 指令：`抽卡排行`
- 功能: 查看本群的抽卡排行榜
